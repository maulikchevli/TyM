Welcome to TyM

An Application that enables you to generate and test models

TyM (pronounced as Tim) is a tool to test Machine Learning Model.

Problem Statement: A software for generation and testing of ML models according to user input. A software for generation and testing of ML models according to user input.

TyM: Test Your Model has two major functions:

  • Generate Model
  • Test Model

In generate model, you have to provide information related to the model you want us to generate and also the training data on which to train the machine learning model. We will do the magic and generate ML models for you. You will be able to view and download your models.
